# vowels [![Support this project][donate-now]][paypal-donations]

A list with the vowels.

## Installation

```sh
$ npm i -g vowels
```

## Example

```js
// Dependencies
var Vowels = require("vowels");

console.log(Vowels);
// => [ 'a', 'e', 'i', 'o', 'u' ]
```

## Documentation

The module exports an array of strings represeting vowels. :sparkles:
